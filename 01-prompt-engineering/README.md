# Prompt Engineering

## Prompt Engineering for Developers

### Two types of LLMs

**Base LLM**

- Predicts next word, based on text training data

_Once upon a time, there was a unicorn_ that lived in a magical forest with all her unicorn friends.

**Instruction Tuned LLM**

- Tries to follow instructions
- Fine-tune on instructions and attempts at following those instructions.
- Works on **RLHF: Reinforcement Learning with Human Feedback**
- Clear, specific instructions (with examples) = Better output
